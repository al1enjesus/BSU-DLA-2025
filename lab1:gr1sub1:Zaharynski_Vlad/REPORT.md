## Задание A. Частотный анализ syslog

### Команда
```bash
cat /var/log/syslog \
  | tr -cs '[:alnum:]' '\n' \
  | tr '[:upper:]' '[:lower:]' \
  | sort \
  | uniq -c \
  | sort -nr \
  | head -n 5
```

### Результат
```
27949 00
15403 ubuntu
15166 linux
15046 2404
11103 08
```

### Вывод
Наиболее часто встречающиеся слова в логе: `00`, `ubuntu`, `linux`. Первое связано с временными метками (`00` минут), остальные с названием дистрибутива и системы.

## Задание B. Неудачные попытки входа

### Команда
```bash
grep -E "Failed|Invalid" /var/log/auth.log  \
  | grep -Eo '([0-9]{1,3}\.){3}[0-9]{1,3}'  \
  | sed -E 's/([0-9]+\.[0-9]+\.[0-9]+\.)[0-9]+/\1x/g'  \
  | sort \
  | uniq -c \ 
  | sort -nr \ 
  | head -n 10
```

### Результат
```
(нет данных)
```

### Вывод
В файле `auth.log` на данный момент не зафиксированы неудачные попытки входа. Это ожидаемо, так как система новая и не подвергалась внешним атакам.

## Задание C. Установки пакетов

### Команда
```bash
grep "install " /var/log/dpkg.log \
  | awk '{print $4}' \
  | sort \ 
  | uniq -c \ 
  | sort -nr \ 
  | head -n 10
```

### Результат
```
2 lxd-installer:all
1 zstd:arm64
1 zlib1g-dev:arm64
1 zlib1g:arm64
1 zip:arm64
1 zerofree:arm64
1 zenity-common:all
1 zenity:arm64
1 yelp-xsl:all
1 yelp:arm64
```

### Вывод
Чаще всего устанавливался пакет `lxd-installer:all` (2 раза). Остальные пакеты были установлены по одному разу — в основном это утилиты и библиотеки (`zlib`, `zip`, `zenity`, `yelp`).
