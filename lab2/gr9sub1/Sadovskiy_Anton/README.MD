## Запуск
Для запуска необходим установленный python3
Чтобы запустить консольное меню:
```
python3 src/main.py src/config.json
```

## Меню
Меню состоит из следующих пунктов:
1. Start workers - запускает воркер процессы по текущему конфигу
2. Stop workers - останваливает воркеров (с ожиданием остановки)
3. Light Mode - переключает воркеров в Light мод
4. Heavy Mode - переключает воркеров в Heavy Mode
5. Reload Config - перезагружает конфиг, не останавливая работу воркеров. При перезагрузке может добавить/удалить воркеров если число воркеров поменялось

## Конфиг
Конфиг представлен в виде JSON файла.
Ключи:
- workers - число воркеров
- log_path - путь к файлу, в который собирать логи
- mode_light - описание Light режима
- mode_heavy - описание Heavy режима
- cpu_affinity - список задаваемых affinity
- nice_values - список nice значений
- restart_limit - количество ретраев и окно времени ретрая

Стоит отметить, что cpu_affinity и cpu_affinity должны совпадать с кол-вом воркеров для корректной работы приложения.
