# Мини-супервизор с воркерами

Процесс-родитель (супервизор), который управляет рабочими процессами (воркерами) с поддержкой graceful shutdown, reload и контроля нагрузки.

## Возможности

### Супервизор
- Запуск N воркеров (N ≥ 2)
- Отслеживание состояния воркеров через SIGCHLD
- Корректный сбор зомби-процессов
- Ограничение частоты рестартов (не более 5 за 30 секунд)

### Обработка сигналов
- **SIGTERM/SIGINT**: Graceful shutdown (ожидание до 5 секунд)
- **SIGHUP**: Graceful reload (перечитывание конфига и перезапуск)
- **SIGUSR1**: Переключение в легкий режим нагрузки
- **SIGUSR2**: Переключение в тяжелый режим нагрузки

### Воркеры
- Имитация работы (циклы вычислений + ожидания)
- Обработка SIGTERM для graceful shutdown
- Обработка SIGUSR1/SIGUSR2 для смены режима
- Диагностический вывод с статистикой

## Формат конфигурации

Конфигурационный файл использует простой формат `KEY=VALUE`:

```bash
workers=3
mode_heavy_work_us=9000
mode_heavy_sleep_us=1000
mode_light_work_us=2000
mode_light_sleep_us=8000
