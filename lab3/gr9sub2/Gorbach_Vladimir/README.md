# Supervisor-Worker Project

Это утилита `pstat` на C++ для извлечения и отображения ключевых метрик процесса из системы `/proc`.

## Зависимости

Для сборки и запуска проекта вам понадобятся:

- `make`
- `g++` (компилятор C++)
- `Любой дистрибутив Linux` (протестировано на Ubuntu 22.04)

## Сборка

Для сборки проекта используйте `Makefile`. Бинарные файлы будут созданы в директории `bin/`.

## Использование

### Запуск

Для компиляции и запуска выполните:

```bash
make run pid=some_pid
```

### Тестирование

Для тестирования выполните:

```bash
make test
```

### Очистка

Для удаления скомпилированных файлов выполните:

```bash
make clean
```
