# pstat - Утилита статистики процессов

Утилита для Linux, отображающая детальную информацию о процессах из файловой системы `/proc`.

## Возможности

- Читает информацию о процессах из `/proc/<pid>/stat`, `/proc/<pid>/status`, `/proc/<pid>/io`
- Отображает комплексную статистику процессов включая:
  - Имя процесса и состояние
  - PID родителя и количество потоков
  - Время CPU (пользовательское и системное)
  - Переключения контекста (добровольные и принудительные)
  - Использование памяти (VmRSS, RssAnon, RssFile)
  - Статистика I/O (прочитанные/записанные байты)
- Автоматическое форматирование единиц измерения (KiB/MiB/GiB)

## Требования

- Операционная система Linux с файловой системой `/proc`
- Утилита-калькулятор `bc`
- Оболочка Bash

## Установка

```bash
# Сделать pstat исполняемым
chmod +x ./src/pstat
```

При желании можно скапировать в /bin/bash
```bash
cp src/pstat -t /usr/local/bin/
```
Или временно добавить в $PATH (запускать находясь в этой директории)
```bash
export PATH=$PATH:$PWD/src
```

## Использование 
```bash
# Базовое использование
./pstat <pid>

# Пример: Показать статистику для текущей оболочки
./pstat $$

# Пример: Показать статистику для процесса с PID 1234
./pstat 1234
```

## Запуск демонстрации
```bash
# Запустить демонстрационный скрипт
chmod +x run.sh
./run.sh
```

## Поля вывода

- **Name**: Имя процесса
- **State**: Состояние процесса (R, S, D, Z, и т.д.)
- **CPU time**: Общее время использования CPU
- **PPid**: ID родительского процесса
- **utime/stime**: Пользовательское и системное время в тиках
- **Threads**: Количество потоков
- **Voluntary/Nonvoluntary switches**: Переключения контекста
- **VmRSS**: Resident Set Size
- **RssAnon/RssFile**: Анонимная и файловая память
- **Read/Write bytes**: Операции ввода-вывода

## Инструменты для сравнения

Утилиту можно сравнивать с:
- `ps` - статус процессов
- `top` - мониторинг процессов в реальном времени
- `pidstat` - детальная статистика процессов
