# Лабораторная работа №3: /proc и утилита pstat

## Описание
В рамках лабораторной работы разработана утилита `pstat` на языке Python. Она считывает информацию о процессе из файловой системы `/proc` (файлы `status`, `stat`, `io`) и выводит сводку по потреблению памяти, процессорному времени и операциям ввода-вывода.

## Ограничения окружения

*   **Операционная система:** Утилита предназначена только для **Linux** (включая WSL2), так как зависит от файловой системы `/proc`. На macOS или Windows (без WSL) работать не будет.
*   **Права доступа:** Для чтения `/proc/<pid>/io` (статистика ввода-вывода) могут потребоваться права суперпользователя (root) или владение процессом. Если прав нет, поля I/O будут помечены как `N/A` (ошибка доступа будет выведена в stderr).
*   **Состояние процесса:** Так как чтение файлов не атомарно, процесс может завершиться во время работы утилиты, что приведет к ошибке.

## Установка и подготовка

Требуется установленный Python 3.

1.  Поместите код утилиты в файл `src/pstat.py`.
2.  Сделайте файл исполняемым:
    ```bash
    chmod +x src/pstat.py
    ```
