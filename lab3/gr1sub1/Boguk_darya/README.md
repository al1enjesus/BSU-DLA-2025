# Лабораторная 3
## Требования
Ubuntu 22.04
Компилятор GCC 11.04

## Сборка
использовать Makefile
```
make build
```
Соберет необходимые бинарники в папку bin

## Тестирование и запуск
Для тестирования можно запустить спящий поток или поток иммитирующий работу, например так
```
while true; do : ; done & pid=$!
echo "PID $pid"
```
Зная pid запустим команду
```
make run pid=$pid
```
