# pstat

Утилита для вывода основных метрик процесса из `/proc`.

## Описание

`pstat` — это Python-скрипт, который анализирует информацию о процессе по его PID, используя данные из:

- `/proc/<pid>/stat`
- `/proc/<pid>/status`
- `/proc/<pid>/io`

Выводит информацию о CPU-времени, памяти, потоках, переключениях контекста и IO-операциях.

## Запуск

```bash
cd src
python3 proc_info.py <PID>
```

## Пример

`python3 proc_info.py 1234`
