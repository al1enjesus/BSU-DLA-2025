# Лабораторная 3 — /proc, собственные метрики и диагностика

Рабочая среда: WSL2, Ubuntu 24.04.

## Запуск
Для запуска Python-утилиты используйте run.sh или выполните скрипт напрямую.

Примеры:
```bash
# сделать исполняемым (если ещё не сделали)
chmod +x src/pstat.py run.sh
# запустить вручную
./src/pstat.py <pid>
# или через run.sh
./run.sh <pid>
```

## Что находится в репозитории
- `src/pstat.py` — основная утилита
- `REPORT.MD` — отчёт по лабораторной работе (цели, шаги, ответы на вопросы)
- `run.sh` — простая обёртка для запуска
