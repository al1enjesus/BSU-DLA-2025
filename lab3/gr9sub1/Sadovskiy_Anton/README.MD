# Установка

## Вариант 1 - deploy.sh

### Установка
```
cd src
chmod +x deploy.sh clear_env.sh
./deploy.sh
```

Это сделает pstat исполняемым, добавит symlink в одну из стандартного набора директорий с исполняемыми файлами, которые хранятся в PATH:
- "/usr/local/bin"
- "/usr/bin"
- "~/.local/bin"

А также сохранит метаданные о расположении бинаря для clear_env.sh в ~/.config/pstat/meta

### Проверка
```
pstat $$
```

### Удаление
Когда нужно будет удалить симлинк, можно просто использовать скрипт src/clear_env.sh
```
./clear_env.sh

```

## Вариант 2 - ручная установка
Если не пользоваться deploy.sh, можно просто сделать pstat исполняемым:
```
chmode +x src/pstat
```

И при каждом вызове прописывать путь к pstat:
```
./src/pstat <pid>
```
