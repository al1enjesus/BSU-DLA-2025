# README — Лабораторная 3 (/proc, собственные метрики и диагностика)

## Требования

- ОС: Linux (тестировалось на Ubuntu 24.04.3, ядро 6.14.0-32-generic)
- Python 3.10+
- Дополнительно для части E*:
  - `strace`
  - `perf` (требует root или альтернативный запуск без root при наличии поддерживаемой версии)

## Установка зависимостей:

sudo apt update
sudo apt install -y python3 python3-venv python3-pip strace linux-tools-common linux-tools-generic

## Запуск для любого процесса:
./src/pstat.py <pid>
Например, посмотреть на свой bash:
./src/pstat.py $$   # $$ — PID текущего shell
