# Задание A

## Цель
Подсчитать 5 самых частых “слов” в `/var/log/syslog` (нормализовать регистр, отделить слова).

## Шаги/решения
cat /var/log/syslog \
  | tr -cs '[:alnum:]' '\n' \
  | tr '[:upper:]' '[:lower:]' \
  | sort \
  | uniq -c \
  | sort -nr \
  | head -n 5 > logs/syslog_top.txt

Сразу же сохранял логи в файлы.

  27566 by
  26389 hp
  26288 timofey

# Задание Б

## Цель
Найти строки с `Failed` или `Invalid` в `/var/log/auth.log`.
Извлечь IP‑адреса, посчитать TOP‑10 источников. 
Замаскировать последний октет IP (например, `192.168.0.15 → 192.168.0.x`) с помощью `sed`.

## Шаги/решения
grep 'Failed\|Invalid\|incorrect' /var/log/auth.log 
| grep -o '\([0-9]\{1,3\}\.\)\{3\}[0-9]\{1,3\}' 
| sed 's/\([0-9]\+\.[0-9]\+\.[0-9]\+\.\)[0-9]\+/\1x/' 
| sort 
| uniq -c 
| sort -nr 
| head -n 10

# Задание B

## Цель
По `/var/log/dpkg.log` посчитать, какие пакеты устанавливались чаще всего (события `install`).
Вывести TOP‑10 пакетов с количеством установок.

## Шаги/решения
grep " install " logs/dpkg.log \
| awk '{print $4}' \
| sed -E 's/:.*$//' \
| sort \
| uniq -c \
| sort -nr \
| head -n 10

## Вывод

>   1 tree \
>   1 sl \
>   1 libonig5 \
>   1 libjq1 \
>   1 jq \
