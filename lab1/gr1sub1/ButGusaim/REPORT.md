## Задание A. Частотный анализ syslog

```bash
cat /var/log/syslog \
  | tr -cs '[:alnum:]' '\n' \
  | tr '[:upper:]' '[:lower:]' \
  | sort \
  | uniq -c \
  | sort -nr \
  | head -n 5

## Результат

3120 kernel
2985 systemd
2740 error
2601 info
2480 network

## Задание B. Поиск неудачных попыток входа

grep -E "Failed|Invalid" /var/log/auth.log \
  | grep -Eo '([0-9]{1,3}\.){3}[0-9]{1,3}' \
  | sed -E 's/([0-9]+\.[0-9]+\.[0-9]+)\.[0-9]+/\1.x/g' \
  | sort \
  | uniq -c \
  | sort -nr \
  | head -n 10

---

7  10.0.2.x
3  172.16.1.x
