## Задание А: Частотный анализ слов в syslog (TOP‑5)

```
cat /var/log/syslog \
| tr -cs '[:alnum:]' '\n' \
| tr '[:upper:]' '[:lower:]' \
| sort \
| uniq -c \
| sort -nr \
| head -n 5
```

Шаги:
```
чтение содержимого syslog
заменяем все символы кроме буквенно-цифровых на новые строки
приводим к нижнему регистру
сортируем по алфавиту
подсчет уникальных вхождений
сортируем по кол-ву в обратном порядке
берем первые 5 записей
```

Результаты:
```
1620072 22
929545 onevoker
929530 sep
927312 session
927228 usr
```

Выводы:
Часто в логах встречается какой-то юзернейм, сентябрь и на первом месте число 22(сегодня 22 сентября)

## Задание Б: Неудачные попытки входа (auth.log)

```
cat /var/log/auth.log \
| grep -E "Failed|Invalid" \
| grep -oE '[0-9]+\.[0-9]+\.[0-9]+\.[0-9]+' \
| sed -E 's/([0-9]+\.[0-9]+\.[0-9]+\.)[0-9]+/\1x/g' \
| sort \
| uniq -c \
| sort -nr \
| head -10
```

Шаги:
```
поиск по содержимому файла 
фильтр по Failed, Invalid
извлечение IP-адреса(вывод только совпадения благодаря -o)
замена последнего октанта
сортировка
подсчет повторов
сортировка по числу в обратном порядке
вывод первых 10
```

Результаты:
2 192.168.1.x

Выводы:
Всего 2 неудачных попытки из подсети 192.168.1.x

## Задание В: Установки пакетов (dpkg.log)

```
grep "status installed" /var/log/dpkg.log* \
| awk '{print $5}' \
| sort \
| uniq -c \
| sort -nr \
| head -10
```


Шаги
```
поиск по содержимому файла
извлечение нужного поля(просмотрел файл, тут имя пакета)
сортировка по алфавиту
подсчет повторов
сортировка по числу в обратном порядке
вывод первых 10
```
