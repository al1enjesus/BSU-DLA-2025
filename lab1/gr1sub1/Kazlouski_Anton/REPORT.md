## Задание А: Частотный анализ слов в syslog

**Цель:** Найти 5 самых частых слов в системном логе.

**Команда:**
```bash
cat /var/log/syslog 
| tr -cs '[:alnum:]' '\n' 
| tr '[:upper:]' '[:lower:]' 
| sort 
| uniq -c 
| sort -nr 
| head -n 5
```

Пояснения:
cat - чтение файла
tr -cs '[:alnum:]' '\n' - замена всех небуквенно-цифровых символов на перевод строки (разделение слов)
tr '[:upper:]' '[:lower:]' - приведение к нижнему регистру
sort - сортировка слов
uniq -c - подсчёт уникальных вхождений
sort -nr - сортировка по убыванию количества
head -n 5 - вывод первых 5 строк

Вывод скрипта:
```
8878 00
5101 Ubuntu
4796 server
4360 2025
4360 09
```

| grep -Eo "([0-9]{1,3}\.){3}[0-9]{1,3}" 
| sort | uniq -c | sort -nr 
| head -n 10

grep -E '(Failed|Invalid)' - поиск строк с неудачными попытками

sed -E 's/([0-9]+\.[0-9]+\.[0-9]+\.)[0-9]+/\1x/g' - маскировка последнего октета
