# Лабораторная работа №4: Системные вызовы

## Воспроизводимость

### Сборка

Для сборки всех компонентов (библиотека `libsyscall_spy.so`, исполняемые файлы `benchmark`, `hello_static` и `mytracer`) выполните:

```bash
make
```

### Запуск экспериментов

Для запуска экспериментов и сохранения их вывода в каталог `logs/` используйте следующие команды:

#### Задание A (LD_PRELOAD)
```bash
# Эксперимент с ls
make run_ls

# Эксперимент с cat
make run_cat

# Эксперимент с grep
make run_grep

# Эксперимент со статической программой
make run_static_test
```

#### Задание C* (ptrace)
Для трассировки программы (например, `/bin/ls`) с помощью `mytracer`:
```bash
./bin/mytracer /bin/ls
```

### Очистка

Для удаления сгенерированных файлов и логов выполните:

```bash
make clean
```
