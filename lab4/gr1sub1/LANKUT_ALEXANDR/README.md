# Лабораторная работа 4: Исследование системных вызовов Linux

## Системные требования
- ОС Linux/WSL2
- gcc
- make
- strace
- perf (опционально)

## Сборка и запуск
```bash
# Сборка всех компонентов
make

# Запуск всех тестов
./run.sh

# Очистка
make clean
```

## Структура проекта
- src/syscall_spy.c - библиотека перехвата системных вызовов
- src/benchmark.c - измерение производительности
- src/static_test.c - тест статической компиляции
- Makefile - сборка проекта
- run.sh - запуск тестов

## Проверка результатов
После выполнения тестов результаты будут в директории logs/:
- ls_spy.log - логи перехвата ls
- cat_spy.log - логи перехвата cat
- grep_spy.log - логи перехвата grep
- static_test.log - результаты теста статической компиляции
- perf_stat_detailed.log - результаты измерения производительности
