# Лабораторная работа №4
## Анализ системных вызовов и CPU/IO поведения программ (ls, cat, grep)

### Как запустить
1. Убедитесь, что у вас установлены `strace` и `perf`:
   ```bash
   sudo apt install strace linux-tools-common linux-tools-generic
   ```

2. Сделайте скрипт исполняемым:
   ```bash
   chmod +x run.sh
   ```

3. Запустите скрипт:
   ```bash
   ./run.sh
   ```
   или для полного анализа с perf:
   ```bash
   sudo ./run.sh
   ```

### Структура проекта
```
lab4_cpu_io_analysis/
├── run.sh              # скрипт для запуска эксперимента
├── README.md           # инстркуция
├── REPORT.md           # отчет
├── src/                # исходники
└── logs/               # результат 
```

### Примечания
- Если `perf` недоступен без root, запустите скрипт через `sudo`.
- Все результаты находятся в `logs/`.
