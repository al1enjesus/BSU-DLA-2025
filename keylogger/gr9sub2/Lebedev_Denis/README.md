# Keyboard Logger Kernel Module

Модуль ядра Linux для логирования нажатий клавиш в файл.

## Описание

Этот модуль ядра перехватывает все нажатия клавиш и записывает их в файл `/root/keyboard_log`. Модуль использует:

- Уведомления ядра о событиях клавиатуры
- Рабочие очереди (workqueues) для асинхронной записи
- Буферизацию для минимизации операций ввода-вывода

**Внимание**: Этот код предназначен только для образовательных целей. Использование кейлоггеров без явного согласия пользователя может быть незаконным.

## Сборка и запуск

1. Убедитесь, что установлены необходимы инструменты:

```bash
make check
```

2. Скомпилируйте модуль:

```bash
make
```

3. Загрузите модуль:

```bash
sudo insmod keylogger.ko
```

4. Проверьте лог-файл:

```bash
sudo cat /root/keyboard_log
```

5. Выгрузите модуль:

```bash
sudo rmmod keylogger
```

## Особенности

- Логи записываются в файл `/root/keyboard_log`
- Специальные клавиши (Shift, Ctrl, Enter и др.) обозначаются в формате `_KEYNAME_`
- Поддерживается распознавание символов с учетом регистра (Shift)
- Используется двойная буферизация для минимизации потерь данных

## Требования

- Ядро Linux 2.6+
- Заголовочные файлы ядра
- Права root для загрузки модуля
